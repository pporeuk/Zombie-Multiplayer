# Zombie-Multiplayer


## 1. 프로젝트 개요
소문난 명강의 : 레트로의 유니티 게임 프로그래밍 에센스(개정판) 책을 바탕으로 4장의 좀비 서바이벌 게임을 구현한 프로젝트입니다.

- **개발 환경:** Unity, C#
- **플랫폼:** Windows (PC)

<br>

## 2. 주요 기능
- **멀티플레이 시스템:** 로비 접속, 방 생성 및 참가, 최대 4인 동시 플레이
- **상태 동기화:** 플레이어의 이동, 회전, 애니메이션 및 체력 실시간 동기화
- **좀비 AI (FSM):** 유한 상태 기계를 활용한 좀비의 배회, 타겟 인식, 추적, 공격 패턴 구현
- **무기 및 전투:** 마우스 우클릭으로 사격, 랜덤한 위치에 드랍되는 아이템(장전, hp회복)을 습득

<br>

## 3. 주요 스크립트 구조 및 역할
직접 작성한 핵심 로직은 `Assets/Scripts` 폴더 내에 기능별로 분류되어 있습니다.

- **`Network/`**: 네트워크 연결 및 멀티플레이 세션 관리
  - `NetworkManager.cs`: Photon 서버 연결, 방 생성 및 로비 UI 관리
- **`Player/`**: 플레이어 조작 및 상태 관리
  - `PlayerController.cs`: 키보드/마우스 입력에 따른 이동 및 시점 회전
  - `PlayerHealth.cs`: 체력 관리 및 데미지, 사망 동기화 (RPC 활용)
- **`Enemy/`**: 좀비 AI 로직
  - `ZombieAI.cs`: NavMeshAgent를 활용한 플레이어 추적 및 공격 로직
- **`Manager/`**: 게임 전체 흐름 관리
  - `GameManager.cs`: 웨이브 시스템 및 좀비 스폰, 게임 오버 상태 관리
